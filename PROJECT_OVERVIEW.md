## ✨ What It Does

1. **Analyzes Legal Documents**
   - Extracts and categorizes clauses
   - Identifies parties, dates, amounts
   - Assesses risk levels
   - Generates recommendations

2. **Provides 3 Views**
   - 📊 Visual cards with stats
   - 💾 Clean JSON output
   - 🎨 Highlighted text

3. **Exports Results**
   - Copy JSON to clipboard
   - Download as JSON file
   - Ready for further processing

## 🎯 Interview Talking Points

**"Why this project?"**
> "I wanted to demonstrate structured output generation and domain-specific parsing. Legal documents are perfect because they have consistent patterns but enough variation to be challenging. Plus, it solves a real problem - contract review is time-consuming and expensive."

**"How does it work?"**
> "It's a pattern-matching NLP system. I tokenize documents into sentences, match them against clause patterns using keywords, extract entities with regex, calculate risk scores, and output structured JSON. All client-side JavaScript, no external dependencies."

**"What would you improve?"**
> "I'd add PDF parsing with OCR, document comparison features, optional AI API integration for enhanced analysis, and a clause template library. Also maybe a browser extension version for real-time analysis."

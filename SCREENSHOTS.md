# Screenshots & Demo Guide

## Taking Screenshots for GitHub

To make your GitHub profile stand out, add screenshots to your README:

1. **Open the project** in your browser (open `index.html`)

2. **Load a sample document** (click "Load NDA" or "Load Service Agreement")

3. **Click "Analyze Document"**

4. **Take screenshots** of:
   - The full interface showing the visual analysis
   - The JSON output tab
   - The highlighted text tab

5. **Create a `screenshots` folder** in your project:
   ```bash
   mkdir screenshots
   ```

6. **Save your screenshots** as:
   - `screenshot-main.png` - Main visual analysis view
   - `screenshot-json.png` - JSON output view
   - `screenshot-highlighted.png` - Highlighted text view

7. **Update README.md** to include the screenshots:
   ```markdown
   ## 📸 Screenshots
   
   ### Visual Analysis
   ![Visual Analysis](screenshots/screenshot-main.png)
   
   ### JSON Output
   ![JSON Output](screenshots/screenshot-json.png)
   
   ### Highlighted Text
   ![Highlighted Text](screenshots/screenshot-highlighted.png)
   ```

## Demo Video (Optional)

Consider recording a short demo video showing:
1. Opening the application
2. Loading a sample document
3. Analyzing it
4. Switching between tabs
5. Downloading JSON output

Upload to YouTube or create a GIF using tools like:
- **LICEcap** (free, Windows/macOS)
- **Kap** (free, macOS)
- **ScreenToGif** (free, Windows)

## Live Demo (GitHub Pages)

To host your project live for free:

1. Push your code to GitHub
2. Go to repository Settings
3. Scroll to "Pages" section
4. Select "main" branch as source
5. Click Save

Your project will be live at:
`https://YOUR_USERNAME.github.io/legal-ai-assistant/`

Add this to your README:
```markdown
## 🌐 Live Demo

Try it live: [https://YOUR_USERNAME.github.io/legal-ai-assistant/](https://YOUR_USERNAME.github.io/legal-ai-assistant/)
```

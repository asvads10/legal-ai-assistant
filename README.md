# ⚖️ Legal AI Assistant

A lightweight legal document analysis tool demonstrating structured output parsing, clause extraction, and risk assessment - **100% free, runs entirely in your browser.**

## 🎯 What It Does

- **Extract Clauses**: Automatically identifies and categorizes legal clauses
- **Risk Analysis**: Flags potential risks and unfavorable terms
- **Structured Output**: Generates valid JSON from unstructured legal text
- **Entity Recognition**: Identifies parties, dates, amounts, and obligations

## ✨ Features

- 🚀 No API keys required - runs entirely client-side
- 📱 Clean, professional interface
- 🎨 Real-time clause highlighting
- 📊 Visual risk assessment
- 💾 Export results as JSON
- 🔍 Pattern-based extraction (no external AI needed)

## 🛠️ Tech Stack

- Pure HTML/CSS/JavaScript
- No dependencies, no build tools
- Pattern matching & NLP techniques
- Regex-based clause detection

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/legal-ai-assistant.git
cd legal-ai-assistant
```

2. Open `index.html` in your browser:
```bash
open index.html  # macOS
# or
start index.html  # Windows
# or just double-click the file
```

That's it! No installation, no configuration, no API keys.

## 📖 How It Works

The assistant uses pattern matching and linguistic analysis to:

1. **Tokenize** legal text into sentences and clauses
2. **Classify** clauses using keyword patterns and structure
3. **Extract** entities like parties, dates, monetary amounts
4. **Assess** risk levels based on clause types and language
5. **Structure** everything into clean JSON output

## 🎓 Example Use Cases

- Review NDAs and identify confidentiality obligations
- Extract key terms from service agreements
- Flag potentially unfavorable clauses in contracts
- Parse employment agreements for non-compete clauses
